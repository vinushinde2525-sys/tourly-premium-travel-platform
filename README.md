## 📖 About the Project

> **Tourly** is a modern full-stack travel platform built with **React, Vite, and Express.js** — designed to deliver an engaging, visually rich travel discovery experience through curated destinations, travel packages, and dynamic content.

Traditional travel websites are often cluttered and poorly optimised for different devices. **Tourly** solves this with a clean, fast, and fully responsive interface backed by a scalable REST API architecture.

This project was built as a **portfolio-grade application** simulating a production-ready travel ecosystem, demonstrating reusable component architecture, smart filtering, and modular full-stack design using static JSON data with a real Express backend.

  │

  │
  ▼

  │
  ▼

## 📂 Project Structure

```bash
tourly/
│
├── 📁 client/                          # React + Vite SPA
│   └── src/
│       ├── 📁 components/             # Reusable UI components
│       │   ├── layout/                # Navbar, Footer, PageWrapper
│       │   ├── sections/              # Hero, Destinations, Testimonials
│       │   └── ui/                    # Buttons, Cards, Modals
│       │
│       ├── 📁 pages/                  # Route-level page components
│       │   ├── Home.jsx
│       │   ├── Destinations.jsx
│       │   ├── DestinationDetail.jsx
│       │   ├── Packages.jsx
│       │   ├── PackageDetail.jsx
│       │   ├── Contact.jsx
│       │   └── NotFound.jsx
│       │
│       ├── 📁 hooks/                  # Custom React hooks
│       ├── 📁 utils/                  # Centralised API helper functions
│       └── App.jsx                    # Root component & router config
│
└── 📁 server/                          # Node.js + Express REST API
    ├── 📁 data/                        # Static JSON data (destinations, packages)
    ├── server.js                       # Express app entry point
    └── package.json
```

## ⚙️ Installation

**Step 1 — Clone the repository**

```bash
git clone https://github.com/vinushinde2525-sys/tourly-premium-travel-platform.git
cd tourly
```

**Step 2 — Install dependencies**

```bash
npm install

cd client
npm install

cd ../server
npm install
```

**Step 3 — Run the project**

```bash
npm run dev
```

✅ Frontend: `http://localhost:3000`
✅ Backend: `http://localhost:5000/api`
